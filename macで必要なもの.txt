## ランタイム / バージョンマネージャ
- mise … nix(home-manager) 管理。node / python / rust / bun / uv / just を統合
- volta … brew 管理。node / pnpm / yarn
- Go（~/go/bin — go install 用）
- rbenv … 今回未検出（mise の ruby 統合で代替可）
- npm global packages（volta 配下）

## 備考
- すでに brew/nix 管理（手動不要）: mise(nix), volta(brew), opencode(brew), node 系
- volta と mise が並走中 → どちらかに寄せると管理が楽
- deno / bun / uv / rust は standalone と mise の二重管理 → mise に寄せれば nix で自動復元できる
- Claude Code は現在 ~/.local/bin のネイティブ版（旧: npm global から移行済み）
- ~/bin, ~/dotfiles/scripts は個人スクリプト用（管理不要）
